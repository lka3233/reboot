1. Необходимо доработать приложение ГСМ следующим функционалом:
 - добавить систему команд: ADD, CALC, TYPE (ADD - добавление информации об авто, CALC - вывод итоговой инфомации, TYPE - вывод информации об отдельном типе авто)
 - хранить стоимость типа товлива в файле
 - хранить расход топлива для типов авто в файле
 - добавить логирование действий (можно сделать позже, после реализации задачи 2).
 * - сделать возможность добавлять и хранить информацию в разрезе каждого дня.
 * - добавить при входе в приложение авторизацию пользователя
 
 2. Необходимо реализовать класс, который будет осуществлять логирование
 действий пользователя в приложении.
 Каждое действие пользователя должно быть записано в файл в формате:
 
 [Дата и время действия] - Имя пользователя (произвольно) - Тип действия
 
 В классе должны быть также реализованы следующие методы:
 - вывод информации в разрезе каждого дня, в разрезе пользователя.
